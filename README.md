# scala-web-bloop-sbt-spring-hello-world

## Description
A minimalist build for springframework
on scala.

Compiled and ran from build server `bloop`.

# Build note
Dependencies must be compatable with jdk8 or less.

## Tech stack
- bloop
- bloop-sbt
- scala

## Docker stack
- eed3si9n/sbt

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
